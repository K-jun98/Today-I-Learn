#### Bean이란?

- Java에서 인스턴스 생성
  - 프로그래머가 직접 인스턴스를 생성함

```java
Book book = new Book();
```
- Bean은 컨테이너가 관리하는 객체
  - 객체의 생명주기를 컨테이너를 관리한다.
  - 객체를 싱클턴으로 만들것인지, 프로토타입으로 만들것인지

즉 컨테이너에 의해 관리되는 객체
